# Video widget media

Drop your "About me" clip here so the sticky bottom-right widget plays it:

- `about-me.mp4`        — the video (landscape / 16:9 recommended)
- `about-me-poster.jpg` — optional still shown before play

These are referenced by the widget at `/media/about-me.mp4` and
`/media/about-me-poster.jpg`. Until added, the widget shows a styled
placeholder with a play button.
